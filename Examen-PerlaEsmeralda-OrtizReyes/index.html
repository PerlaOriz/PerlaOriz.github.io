  1. Comando ls El comando ls se utiliza para listar archivos o directorios en Linux y otros sistemas operativos basados en linux
    • -l Escriba el comando ls -l para listar el contenido del directorio en un formato de tabla con columnas incluidas: permisos de contenido, número de enlaces al contenido, propietario del contenido, propietario del grupo del contenido, tamaño del contenido en bytes, fecha / hora de la última modificación del contenido, nombre de archivo o directorio
    • -a Escribe el comando ls -a para listar archivos o directorios, incluidos archivos o directorios ocultos. En Linux, cualquier cosa que comience con un . se considera un archivo oculto:
  2. Comando touch El comando touch de Linux se usa principalmente para crear archivos vacíos y cambiar marcas de tiempo de archivos o carpetas.
  3. Comando mkdir El comando mkdir se utiliza para crear un directorio especifico.
  4. Comando rm Con “rm” seguido del nombre del archivo se pueden borrar archivos individuales de manera rápida y sencilla a través de la terminal. Con el comando “rm” seguido del nombre de archivo podrás eliminar de manera rápida archivos individuales en Linux.
  5. Comando nano Sus parametros son
    • ctrl + k = Corta una linea.
    • ctrl + w = Buscar texto.
    • ctrl + o = Guardar.
    • ctrl + x = Salir.
